class callback.CallbackManager¶
add(prefix, key, callback, one_shot=True, only_caller=None)¶

Add a callback to the stack for the specified key. If the call is specified as one_shot, it will be removed after being fired

classmethod instance(class_)¶

pending(prefix, key)¶

Return count of callbacks for a given prefix or key or None

process(prefix, key, caller, *args, **keywords)¶

Run through and process all the callbacks for the specified keys
